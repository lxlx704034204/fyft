#fyft

面向webapp开发

涉及技术:

* Maven 构建工程
* Spring Boot 搭建后台系统 版本 1.5.2.RELEASE
* Spring 全家桶，SpringMVC、Spring JDBC

* fyft-core 基础工程，存放工具封装类
* fyft-wx webapp 单页应用。使用 vue.js 框架，使用 require.js 管理模块化文件
* fyft-admin 后台管理
